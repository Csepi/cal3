# Test Layout

- `unit/`: shared unit test fixtures and utility runners.
- `integration/`: cross-service and API integration assets.
- `e2e/`: references for end-to-end scenarios (web/mobile suite is in `/e2e`).
- `security/`: OWASP ZAP and payload dictionaries.
- `load/`: Artillery and k6 smoke load profiles.
- `fixtures/`: deterministic seed fixtures used by tests.
- `api/`: Postman/Newman collection and environment.
