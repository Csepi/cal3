ISSUE FIXED:
The batch scripts were failing with:
  '"node"' is not recognized as an internal or external command

ROOT CAUSE:
Node.js was not in the Windows CMD PATH, even though it was 
installed at: C:\Program Files\nodejs

SOLUTION APPLIED:
All batch scripts now include this line at the beginning:
  set "PATH=C:\Program Files\nodejs;%PATH%"

This adds Node.js to PATH for the duration of the script execution.
