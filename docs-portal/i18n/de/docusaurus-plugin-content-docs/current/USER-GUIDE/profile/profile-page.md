# Profilseite {#profile-page}

Die Profilseite ist das Kontrollzentrum für die Identität, Lokalisierung, das Erscheinungsbild, die Bezeichnungen und das Focus-Verhalten Ihres Kontos.

## So öffnen Sie es {#how-to-open-it}

1. Öffnen Sie den Hauptarbeitsbereich.
2. Öffnen Sie Ihr Benutzermenü.
3. Wählen Sie `Profile`.

![PrimeCal Seite mit vollständigen Profileinstellungen](../../assets/user-guide/profile/profile-settings-full.png)

## Hauptbereiche auf der Seite {#main-areas-on-the-page}

| Bereich | Was Sie kontrollieren | Warum es wichtig ist |
| --- | --- | --- |
| Identität | Benutzername, E-Mail, Vorname, Nachname, Profilbild | Diese Details bestimmen, wie Sie in PrimeCal erscheinen. |
| Gebietsschema | Sprache, Zeitzone, Zeitformat, Wochenstart | Diese Einstellungen beeinflussen jeden Kalender und jedes Aufgabendatum, das Sie lesen. |
| Kalenderstandards | Standardaufgabenkalender und Sichtbarkeitseinstellungen | Nützlich, wenn Sie mit mehreren Kalendern gleichzeitig arbeiten. |
| Veranstaltungsetiketten | Wiederverwendbare Ereignis-Tags | Beschleunigt die Klassifizierung von Ereignissen und die Erstellung von Fokusfiltern. |
| Fokuseinstellungen | Versteckte Live-Fokus-Beschriftungen und Verhalten bei Nicht-Fokus | Hilft Ihnen, die Live-Focus-Ansicht zu beruhigen, ohne überall dieselben Elemente auszublenden. |
| Sicherheit | Passwortänderung | Behält den Kontozugriff unter Ihrer Kontrolle. |
| Aussehen | Themenfarbe und visuelle Vorlieben | Hält den Arbeitsbereich lesbar und konsistent. |

## Fokusspezifische Einstellungen {#focus-specific-settings}

Die wichtigste Fokusoption ist die Liste der Beschriftungen, die in der Live-Fokusansicht ausgeblendet sind.

- Ereignisse mit diesen Bezeichnungen bleiben im Kalender.
- Sie werden weiterhin in der Monats- und Wochenansicht angezeigt, es sei denn, Sie blenden ihren Kalender aus.
- Der Filter ist nützlich für Hintergrundelemente wie Besorgungen, Verwaltungsarbeiten oder passive Erinnerungen.

![PrimeCal Profilseite mit konfigurierten Focus-Label-Filtern](../../assets/user-guide/profile/profile-settings-focus-filter.png)

## Gute Profilgewohnheiten {#good-profile-habits}

- Legen Sie die Zeitzone fest, bevor Sie wichtige Ereignisse erstellen.
- Halten Sie Veranstaltungsetiketten kurz und wiederverwendbar, damit sie leicht gefiltert werden können.
- Gehen Sie sparsam mit versteckten Fokusbezeichnungen um. Zu viele Filter können die Live-Ansicht unübersichtlich machen.
- Überprüfen Sie Ihr Profil noch einmal, nachdem Sie einige Kalender und Routinen erstellt haben.
