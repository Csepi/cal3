# Arbeitsbereich „Aufgaben“. {#tasks-workspace}

PrimeCal hält Aufgaben nah am Kalender, sodass Sie zwischen zeitbasierter Planung und umsetzbarer Arbeit wechseln können, ohne das Produkt wechseln zu müssen.

## So öffnen Sie es {#how-to-open-it}

1. Öffnen Sie den Hauptarbeitsbereich.
2. Wählen Sie `Tasks`.

![PrimeCal Aufgabenarbeitsbereich mit realistischem Familienaufgabenboard](../../assets/user-guide/tasks/tasks-workspace.png)

## Was der Aufgabenarbeitsbereich beinhaltet {#what-the-tasks-workspace-includes}

| Bereich | Wofür es ist |
| --- | --- |
| Aufgabenliste | Das Hauptboard für aktives Arbeiten |
| Filter | Grenzen Sie nach Status, Labels oder Suche ein |
| Schnell erstellen | Fügen Sie eine Aufgabe hinzu, ohne den Arbeitsbereich zu verlassen |
| Detailkomponist | Geben Sie Fälligkeitstermine, Beschriftungen, Notizen und Farben ein |

## Schnell erstellen {#quick-create}

Verwenden Sie die Funktion „Schnell erstellen“, wenn Sie etwas schnell festhalten möchten, bevor Sie es vergessen.

Beispiele:

- Lebensmittelnachsorge
- Schulunterlagen
- Termine zu vereinbaren
- Packen oder Reisevorbereitung

![PrimeCal Aufgaben-Schnellerstellung und Composer-Ansicht](../../assets/user-guide/tasks/task-composer.png)

## Gute Aufgabengewohnheiten {#good-task-habits}

- Verwenden Sie Beschriftungen für Bereiche wie `School`, `Home` oder `Errands`.
- Halten Sie Titel kurz und handlungsorientiert.
- Fügen Sie Fälligkeitstermine nur dann hinzu, wenn es wirklich auf das Timing ankommt.
- Verwenden Sie den Standard-Aufgabenkalender für die aufgabenbezogene Planung, anstatt diese Elemente in normale Ereigniskalender zu mischen.
