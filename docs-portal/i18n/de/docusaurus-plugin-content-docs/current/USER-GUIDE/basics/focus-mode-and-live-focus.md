# Fokusmodus und Live-Fokus {#focus-mode-and-live-focus}

PrimeCal verfügt über zwei verwandte Fokusverhalten:

- `Live Focus` im normalen Timeline-Arbeitsbereich
- `Clean focus mode`, wodurch der größte Teil des umgebenden Chromes entfernt wird und nur der aktuelle Timeline-Kontext übrig bleibt

## Wechseln Sie in den Clean Focus-Modus {#enter-clean-focus-mode}

1. Öffnen Sie `Calendar`.
2. Wechseln Sie bei Bedarf zu `Timeline`.
3. Klicken Sie auf `Clean focus mode`.
4. Klicken Sie auf `Exit focus mode`, wenn Sie den gesamten Arbeitsbereich zurückhaben möchten.

![Live Focus-Zeitleiste mit dem aktuellen Familienereignis](../../assets/user-guide/views/focus-view-live-family-calendar.png)

## Was versteckte Fokusetiketten bewirken {#what-hidden-focus-labels-do}

Verwenden Sie `Profile` -> `Hide labels from LIVE focus`, wenn Sie möchten, dass einige Ereignisse im Kalender, aber außerhalb der Live-Fokus-Oberfläche bleiben.

- Ereignisse mit diesen Bezeichnungen werden weiterhin in der Monatsansicht angezeigt.
- Sie werden weiterhin in der Wochenansicht angezeigt.
- Sie werden von der Live-Focus-Karte und dem Timeline-Vorschaustreifen entfernt.

![Profileinstellung, die Beschriftungen nur im Live-Fokus verbirgt](../../assets/user-guide/profile/profile-settings-focus-filter.png)

## Wie der ruhige Staat aussieht {#how-the-quiet-state-looks}

Wenn das aktuelle Ereignis herausgefiltert wird oder kein passendes Live-Ereignis vorhanden ist, wechselt Focus in den Ruhezustand.

![Fokusmodus bereinigen, ohne dass ein passendes Live-Ereignis angezeigt wird](../../assets/user-guide/views/focus-mode-clean-filtered.png)

## Wann man es verwendet {#when-to-use-it}

- Blenden Sie Beschriftungen wie `routine`, `household` oder `no-focus` aus, wenn diese Elemente die Live-Ansicht überladen.
- Halten Sie die Familienlogistik in der Monats- oder Wochenansicht sichtbar und schützen Sie gleichzeitig die aktuelle Fokusoberfläche.
- Verwenden Sie den Clean-Focus-Modus auf einem zweiten Bildschirm, wenn Sie tagsüber eine vereinfachte Zeitleiste wünschen.
