## Was passiert zuerst? {#what-happens-first}

<div class="pc-guide-flow">
  <article class="pc-guide-flow__item">
    <div class="pc-guide-flow__index">1</div>
<h3>Register</h3>
    <p>Erstellen Sie das Konto mit einem Benutzernamen, einer E-Mail-Adresse und einem Passwort.</p>
  </article>
  <article class="pc-guide-flow__item">
    <div class="pc-guide-flow__index">2</div>
    <h3>Onboarding abschließen</h3>
    <p>Wählen Sie im geführten Assistenten Ihre Sprache, Zeitzone, Wocheneinstellungen und Compliance-Optionen aus.</p>
  </article>
  <article class="pc-guide-flow__item">
    <div class="pc-guide-flow__index">3</div>
    <h3>Erstellen Sie einen echten Kalender</h3>
    <p>Der Standardkalender `Tasks` ist nützlich für die Aufgabenerfassung, die meisten Benutzer sollten jedoch sofort einen normalen Kalender hinzufügen.</p>
  </article>
  <article class="pc-guide-flow__item">
    <div class="pc-guide-flow__index">4</div>
    <h3>Planung starten</h3>
    <p>Erstellen Sie das erste Ereignis und fahren Sie dann mit der Monats-, Wochen- und Fokusansicht im Benutzerhandbuch fort.</p>
  </article>
</div>

:::warnung Erstellen Sie frühzeitig einen normalen Kalender
PrimeCal erstellt automatisch einen privaten `Tasks`-Kalender, sodass der Aufgaben-Arbeitsbereich sofort funktioniert. Dies ersetzt keinen normalen Familien-, Privat- oder Arbeitskalender, daher sollten die meisten Benutzer bei der Ersteinrichtung einen erstellen.
:::
