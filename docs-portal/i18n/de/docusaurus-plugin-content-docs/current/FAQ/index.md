## Benötigen Sie die vollständige Komplettlösung anstelle der kurzen Antwort? {#need-the-full-walkthrough-instead-of-the-short-answer}

- Beginnen Sie mit [Erste Schritte](../GETTING-STARTED/index.md), wenn Sie noch Ihr erstes Konto und Ihre ersten Kalender einrichten.
- Fahren Sie mit der [Benutzerdokumentation](../USER-GUIDE/index.md) fort, wenn Sie die vollständige Schritt-für-Schritt-Anleitung zum Produkt wünschen.
- Verwenden Sie [Entwicklerdokumentation](../DEVELOPER-GUIDE/index.md) nur, wenn Sie API Verträge oder Implementierungsdetails benötigen.
