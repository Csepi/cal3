# Hozzájárulás a PrimeCalendar Dokumentációhoz {#contributing-to-primecalendar-documentation}

## Folyamat {#process}

1. Kirendeltség a `main`-tól.
2. A kanonikus dokumentumok frissítése a `docs/` alatt.
3. Futtassa le a dokumentumok minőségi ellenőrzését.
4. Nyújtsa be a PR-t kontextussal és ellenőrző megjegyzésekkel.

## Felülvizsgálati kritériumok {#review-criteria}

- Műszaki pontosság
- Stílusútmutató megfelelés
- Link integritás
- A metaadatok teljessége

## Problémák bejelentése {#reporting-issues}

- Nyisson meg egy GitHub-problémát a következő címkével: `documentation`.
- Tartalmazza az utat, a problémát és az elvárt viselkedést.
- Csatoljon képernyőképeket a felhasználói felülettel kapcsolatos dokumentumokkal kapcsolatos problémákról.

## Negyedéves karbantartás {#quarterly-maintenance}

- Futtassa újra a dokumentumok leltárát és a hiányelemzést.
- Az elavult oldalak archiválása.
- Frissítse a képernyőképeket és a példákat.
- Frissítse a KPI-ket és az állapot-irányítópult bemeneteit.
