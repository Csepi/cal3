# Feladatok Munkaterület {#tasks-workspace}

A PrimeCal a feladatokat a naptár közelében tartja, így termékváltás nélkül válthat az időalapú tervezés és a tényleges munka között.

## Hogyan lehet megnyitni {#how-to-open-it}

1. Nyissa meg a fő munkaterületet.
2. Válassza a `Tasks` lehetőséget.

![PrimeCal feladatok munkaterület valósághű családi feladattáblával](../../assets/user-guide/tasks/tasks-workspace.png)

## Mit tartalmaz a Tasks munkaterület {#what-the-tasks-workspace-includes}

| Terület | Mire való |
| --- | --- |
| Feladatlista | Az aktív munka főtáblája |
| Szűrők | Szűkítés állapot, címkék vagy keresés alapján |
| Gyors létrehozás | Adjon hozzá egy feladatot a munkaterület elhagyása nélkül |
| Részletes zeneszerző | Töltse ki a határidőket, a címkéket, a megjegyzéseket és a színt |

## Gyors létrehozás {#quick-create}

Használja a gyors létrehozást, ha gyorsan szeretne rögzíteni valamit, mielőtt elfelejtené.

Példák:

- élelmiszerbolt nyomon követése
- iskolai papírmunka
- időpontok egyeztetés szerint
- pakolás vagy utazási előkészület

![PrimeCal feladat gyors létrehozása és zeneszerző nézet](../../assets/user-guide/tasks/task-composer.png)

## Jó munkaszokások {#good-task-habits}

- Használjon címkéket olyan területekhez, mint a `School`, `Home` vagy `Errands`.
- A címek legyenek rövidek és cselekvésorientáltak.
- Csak akkor adjon meg határidőket, ha az időzítés valóban számít.
- Használja az alapértelmezett Feladatok naptárat a feladatokkal kapcsolatos tervezéshez ahelyett, hogy ezeket az elemeket normál eseménynaptárakba keverné.

## Fejlesztői referencia {#developer-reference}

Feladatszerződésekhez és címkeútvonalakhoz használja a [Tasks API](../../DEVELOPER-GUIDE/api-reference/tasks-api.md) elemet.
