1. Project Overview

The Calendar MVP is a full-stack web application designed to provide comprehensive calendar management with modern UI/UX. The application features a React frontend with a NestJS backend, supporting multiple users, calendars, events, and an advanced admin system.

🔐 Authentication

JWT-based auth with role-based access control

📅 Calendar Management

Create, share, and manage multiple calendars

🎯 Event Management

Full CRUD with recurring events support

👑 Admin System

Comprehensive admin panel with statistics

🎨 Modern UI

Glassmorphism design with blue theme

⚡ Real-time

Live data updates via REST API

Core Tables

Users Table

Field Type Constraints Description
id INTEGER PK, AUTO_INCREMENT Primary key
username VARCHAR(255) UNIQUE, NOT NULL Username
email VARCHAR(255) UNIQUE, NOT NULL Email address
password VARCHAR(255) NOT NULL Hashed password
role VARCHAR(20) DEFAULT 'user' observer|user|admin
isActive BOOLEAN DEFAULT true Account status

Events Table

Field Type Constraints Description
id INTEGER PK, AUTO_INCREMENT Primary key
title VARCHAR(300) NOT NULL Event title
startDate DATE NOT NULL Event start date
startTime TIME NULLABLE Event start time
recurrenceType VARCHAR(20) DEFAULT 'none' none|daily|weekly|monthly|yearly
calendarId INTEGER FK → calendars Parent calendar
createdById INTEGER FK → users Event creator

9. Installation & Setup

Prerequisites

  • Node.js (v18+ recommended)
  • npm or yarn package manager
  • Git version control

Quick Start

# Clone repository
git clone https://github.com/Csepi/cal3.git
cd cal3

# Setup Frontend (Terminal 1)
cd frontend
npm install
npm run dev
# Runs on http://localhost:8080

# Setup Backend (Terminal 2)
cd backend-nestjs
npm install
npm run seed        # Seed database
npm run start:dev
# Runs on http://localhost:8081

Environment Configuration

# backend-nestjs/.env
DB_PASSWORD="Enter.Enter"
JWT_SECRET="calendar-secret-key"
PORT=8081

10. Deployment

Production Build

# Frontend
npm run build       # Creates dist/ folder

# Backend
npm run build       # Creates dist/ folder
npm run start:prod  # Production server

Docker Configuration

# Frontend Dockerfile
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
RUN npm run build
EXPOSE 8080
CMD ["npm", "run", "preview"]

# Backend Dockerfile
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
RUN npm run build
EXPOSE 8081
CMD ["npm", "run", "start:prod"]

Production Considerations

  • Database: Migrate from SQLite to PostgreSQL
  • Security: Configure HTTPS and CORS
  • Monitoring: Set up logging and error tracking
  • Performance: Enable compression and caching
