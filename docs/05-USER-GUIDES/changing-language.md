# Changing Language

PrimeCalendar supports:

- English
- Hungarian
- German
- French

## Web Application

1. Open the app and sign in.
2. Use the language selector in the top navigation.
3. Choose your preferred language.
4. The UI updates immediately without reloading.

Your selection is saved to:

- your profile preference
- local browser storage

## Mobile Application

- PrimeCalendar uses your in-app language preference after login.
- Android native widget strings are localized for `en`, `hu`, `de`, `fr`.

## Troubleshooting

- If text does not change, refresh once and sign in again.
- If the selector is missing, verify your account has access to the main app shell.
