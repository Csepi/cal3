# Prerequisites

Last updated: 2026-02-03

[‹ Getting Started](./README.md)

## Supported Platforms
- Windows 10/11
- macOS 13+
- Linux (Ubuntu 22.04+, Debian 12+, RHEL 9+)

## Required Versions
- Node.js 18+ (LTS recommended)
- npm 9+ (or yarn 4+)
- PostgreSQL 12+ or Azure SQL
- Optional: Docker 24+ / Docker Compose v2

## Recommended Tooling
- Git 2.40+
- VS Code + ESLint + TypeScript extensions
