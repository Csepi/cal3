# Environment Variables

Last updated: 2026-02-03

[Back to Getting Started](./README.md)

## Backend Core
- `PORT` (default `8081`)
- `NODE_ENV`
- `CORS_ORIGIN`
- `JWT_SECRET`
- `JWT_EXPIRATION`

## Database
- `DB_TYPE` (`postgres` | `mssql` | `sqlite`)
- `DB_HOST`, `DB_PORT`, `DB_NAME`
- `DB_USERNAME`, `DB_PASSWORD`
- `DB_SSL`, `DB_SSL_REJECT_UNAUTHORIZED`
- `DB_POOL_MAX`, `DB_POOL_MIN`
- `DB_IDLE_TIMEOUT`, `DB_CONNECTION_TIMEOUT`

## Frontend
- `VITE_API_BASE_URL`
- `VITE_PUBLIC_API_BASE`
- `VITE_PUBLIC_BOOKING_BASE`

## Integrations
- Google/Microsoft OAuth credentials
- Smart-home adapter settings
- Notification provider settings
